Teaching


Fall 2018 – INF 553: Foundations and Applications of Data Mining

Data mining and machine learning algorithms for analyzing very large data sets. Emphasis on Map Reduce and case studies.
Additional Material: Lecture slides, example programs, and project reports.

Spring 2017 – CSCI 104: Data Structures and Object Oriented Design

Introduction to standard data structures (linear structures such as linked lists, (balanced) trees, priority queues, and hashtables), using the C++ programming language.
Additional Material: Lecture slides and example programs.
